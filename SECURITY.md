# Security Policy

## Supported Versions

Currently, we only support the latest release of Euler OMR.

| Version | Supported          |
| ------- | ------------------ |
| v1.1.5  | :white_check_mark: |
| < 1.1.5 | :x:                |

## Reporting a Vulnerability

We take the security of Euler OMR seriously. If you find a security vulnerability, please do not open a public issue. Instead, please report it via the following method:

1. **Email:** Reach out to the project maintainers directly.
2. **Details:** Include a detailed description of the vulnerability, steps to reproduce, and potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary. Please give us reasonable time to investigate and mitigate the issue before making any information public.
